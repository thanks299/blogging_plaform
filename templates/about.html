About Our Blogging Platform

Our Mission

Our blogging platform aims to provide a seamless experience for writers and readers. Whether you're sharing your thoughts or diving into insightful articles, our platform makes it easy and enjoyable.

About the Project

This project is developed as a collaborative effort by our team to create a user-friendly blog platform. Our GitHub repository, Blogging Platform Repository, is where the code lives and evolves.

The project focuses on providing essential features like user authentication, post creation, editing, and browsing with a simple, intuitive UI. It leverages Flask for the backend and supports secure authentication, dynamic content management, and more.

Technologies Used

  • Flask (Backend Framework)
  • SQLAlchemy (Database ORM)
  • Jinja2 (Template Engine)
  • Flask-Login (User Authentication)
  • Flask-Mail (Email Integration)
  • HTML5, CSS3, JavaScript (Frontend Technologies)
